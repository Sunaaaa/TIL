#### 1. HTML 문서 기본 구조

```html
<!-- 1. !DOCTYPE : 문서 형식 선언부 -->
<!DOCTYPE html>
<!-- 2. html : 문서 시작과 끝 지정 -->
<!-- 2-1. lang : 스크린 리더, 검색 엔진 필터링 도움 -->
<html lang="ko">

<!-- 3. head : 브라우저에게 건너줄 정보 -->
<head>
    <meta charset="UTF-8">
    <title>인트로 페이지</title>
</head>

<!-- 4. body : 실제 사용자가 볼 내용 -->
<body>

</body>
</html>
```

#### 2-2 요소 (Element)

#### 2-3 Self-closing element

- 닫는 태그가 없는 태그도 존재

  ```html
  <img src="url"/>
  ```

#### 2-3-1 속성 (Attribute)

- id, class, style 속성은 태그와 상관없이 모두 사용 가능

  ```html
  <a href="https://google.com"> </a>
  ```

  - 공백 없음
  - 쌍 따움표
